# Alarm Clock Card

A beautiful alarm clock card for Home Assistant alarms.

## Features

- 🎨 Clean, modern design
- 🌙 24-hour time format
- ⏰ Live countdown timer
- 🔔 Easy alarm control
- 🌈 Status indicators

## Usage

```yaml
type: custom:alarm-clock-card
entity: switch.your_alarm_clock
```
